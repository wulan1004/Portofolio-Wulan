PROFIL

Saya adalah mahasiswa semester 6 jurusan Sistem Informasi di Universitas Persada Indonesia YAI, dengan minat besar di bidang Teknologi Informasi. Saya ingin mengembangkan pengetahuan dan pengalaman saya melalui program magang di perusahaan ini.

Sebagai bekal, saya telah menempuh mata kuliah relevan seperti Dasar-Dasar Pemrograman, Data Mining, Business Intelligence, Web Programming, Database, dan Cyber Security, yang memperkuat kemampuan teknis saya.

Saya yakin dengan kombinasi latar belakang pendidikan dan minat yang mendalam ini dapat memberikan kontribusi positif bagi perkembangan perusahaan. Saya siap untuk terus belajar, berkembang, dan bekerja sama demi mencapai kesuksesan bersama tim.

PROJECT

  • Prediksi Prestasi Belajar Siswa Menggunakan Data Mining
    • Menganalisis data siswa sekolah menengah dari Portugal menggunakan algoritma Decision Tree, Naive Bayes, dan Random Forest.
    • Menggunakan metodologi CRISP-DM untuk pemahaman bisnis, pemrosesan data, pemodelan, evaluasi, hingga deployment.
    • Membuat visualisasi model klasifikasi dan evaluasi hasil akurasi menggunakan RapidMiner Studio.
    • Aktif dalam penyusunan paper ilmiah dan presentasi tim menggunakan Canva.
  • Perancangan Prototipe Portal Karier Digital "Tunakarya" untuk Meningkatkan Akses Informasi Lowongan Kerja dan Pelatihan Daring di Jakarta
    • Portofolio online untuk menampilkan hasil karya, 2023.
    • Menganalisis permasalahan sistem rekrutmen konvensional dan merancang proses bisnis baru yang lebih terintegrasi.
    • Menyusun dokumen spesifikasi teknis proyek, termasuk ruang lingkup, fitur utama, kebutuhan fungsional dan non-fungsional.
    • Mendesain alur use case diagram serta skenario interaksi pengguna berdasarkan peran (pencari kerja, perusahaan, admin, pemerintah).
    • Mengkoordinasi pembuatan sistem rekomendasi lowongan berdasarkan profil pengguna dan fitur pelatihan daring berbasis sertifikasi.
  • Analisis Pengeluaran TI Berbasis Dashboard Interaktif dengan Power BI
    • Mendesain dashboard Power BI untuk menampilkan varians anggaran secara real-time berdasarkan wilayah, area bisnis, dan elemen biaya utama.
    • Melakukan data modeling dan integrasi multi-sumber untuk pengambilan keputusan berbasis data.
    • Mengimplementasikan fitur drill-down serta indikator performa untuk memantau penyimpangan anggaran.
    • Berpartisipasi dalam penyusunan laporan analisis sebagai bagian dari tugas studi kasus.
  • Desain dan Pengembangan Website E-Commerce "Street Style Clothing"
    • Mendesain struktur halaman lengkap: Home, Shop, Blog, Styling Tips, Size Guide, Reviews, dan Checkout.
    • Mengintegrasikan konten visual seperti gambar produk, ulasan pelanggan, dan elemen interaktif tombol “Add to Cart” serta “Live Chat”.
    • Mengimplementasikan tabel panduan ukuran dan sistem form checkout dengan berbagai pilihan metode pembayaran.
    • Membuat blog fashion musiman serta panduan gaya berpakaian dan aksesori untuk pengguna.
  • Desain dan Pengembangan Website E-Commerce "Melomance Street"
    • Mendesain antarmuka pengguna (UI) yang menarik dan responsif untuk pengalaman berbelanja yang nyaman.
    • Mengimplementasikan layout halaman Home, Product Catalog, Review Section, Loyalty Program, dan Checkout Form secara terstruktur.
    • Menambahkan fitur interaktif seperti tombol "Add to Cart", sistem live chat, dan form checkout sederhana.
    • Menyusun struktur navigasi dan sistem menu untuk akses cepat ke berbagai kategori produk.
